# Test Your Knowledge (Developer Edition)
10 Question Console Quiz Game about the game's programmer. 
This is a modified, and overall upgraded, version of the Developer Fact Quiz Game. 
Modifications made:
- ❓ Added five (5) more questions to the game and changed a few of the original five (5) questions.  
- 💻 Implemented a clear screen feature using the system("cls"); Code in Visual C++ to refresh the console screen after each question.
- 🎖 Integrated a point system where each question is worth 100 times its number (e.g. Question 6 is worth 600 points).
- 🕹 Utilized the system("cls"); Visual C++ code to print Current Score, Points added/deducted/ and "Game Over" text on their own screen.
